# CAFE_SALES_EXCEL

Description:
Developed an interactive Excel dashboard to analyze café sales performance across multiple dimensions such as item category, location, and payment method. The dashboard provides clear insights into total transactions, quantity sold, and total revenue to support data-driven business decisions.

Key Features:
Designed dynamic charts to visualize item-wise, day-wise, and month-wise sales trends.
Integrated interactive filters for month and day selection to enhance user interactivity.
Displayed KPIs such as total transactions, total quantity sold, and total revenue.
Analyzed sales distribution by location (in-store, takeaway) and payment methods (cash, card, digital wallet).
Built an engaging and professional dashboard layout using Excel charts, slicers, and pivot tables.  

Tools & Skills Used:
Microsoft Excel, Pivot Tables, Charts, Slicers, Data Cleaning, Data Visualization, Dashboard Design, Business Analysis.

Outcome:
Delivered a visually appealing and insight-rich dashboard that helps café management monitor sales performance, identify top-selling items, and understand customer purchasing behavior effectively.
