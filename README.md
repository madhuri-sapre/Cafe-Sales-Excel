# Project Title:
Cafe_Sales_Excel
 
# Description
The Cafe Sales Dashboard provides a comprehensive view of café performance by analyzing key sales metrics such as total transactions, total quantity sold, and total revenue. It helps business owners and managers track item-wise, month-wise, and day-wise sales trends, along with customer payment preferences and sales distribution by location.


# Objectives
•	To analyze café sales data and identify trends across products, months, and locations. <br/>
•	To visualize key performance indicators (KPIs) for data-driven decision-making.  <br/>
•	To understand customer behavior and payment method preferences.  <br/>


# Key Insights & Metrics
•	Total Transactions: 9,039  <br/>
•	Total Quantity Sold: 27,370  <br/>
•	Total Revenue: ₹80,779  <br/>
•	Top-Selling Items: Sandwich, Coffee, Tea, Pastry  <br/>
•	Highest Sales Day: Saturday  <br/>
•	Most Preferred Payment Mode: Credit Card (32%)  <br/>
•	Sales by Location: 60% In-store | 40% Takeaway  <br/>


# Dashboard Features
## Month-wise Sales Trend: 
Line chart showing sales growth throughout the year.  <br/>

## Item-wise Performance: 
Bar chart identifying best and least-selling products.  <br/>

## Day-wise Analysis: 
Visual insights into which days drive the most revenue.  <br/>

## Location-wise Comparison: 
Pie chart highlighting In-store vs Takeaway contribution.  <br/>

## Payment Method Distribution: P
ercentage breakdown of credit card, cash, and digital wallet usage.  <br/>
